# 一、基本格式

```html
<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
<base target="_blank" />
</head>

<body>
The content of the document......
</body>

</html>
```

```html
}

    color:white;

    text-align:center;
   padding:5px;

</head>
```
